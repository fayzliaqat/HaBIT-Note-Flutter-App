# 📝 HaBIT Note - Feature-Rich Flutter Application

A clean, modern, and cross-platform mobile application developed using **Flutter** and **Dart** for the Mobile Application Development curriculum under **Sir Muhammad Rizwan**. 

The application serves as a productivity ecosystem, combining comprehensive user authentication, structured interactive onboarding, and efficient local management of daily personal notes and task execution matrices.

## 🚀 App Flow & Implemented Screens
The application architecture comprises a fully integrated state workflow with the following views:
1. **Introduction Gateway:** Implements a dynamic Splash Screen leading into a multi-stage progressive Onboarding Screen layout.
2. **Secure Authentication Layouts:** Custom built screens for User Registration, Secure Login portal, and Account Recovery (Forgot Password module).
3. **Core Dashboard Architecture:** An interactive HomeScreen equipped with a customized Navigation Drawer, dynamic "Help" and "Me" account profile tabs.
4. **Productivity Toolkits:** Interactive modules featuring a validation-checked "Add Note" canvas and an optimized "Add To-Do" execution layer.
5. **System Utility Contexts:** Dedicated "About Screen" info panels along with context-triggered Log Out operational dialog overlays.

## 🛠️ Engineering Stack & Technical Fundamentals
* **Framework:** Flutter (Cross-Platform Mobile UI SDK)
* **Language:** Dart (Object-Oriented Programming)
* **UI Design Patterns:** Material Design Components, Explicit Form Text Validation, Layout Columns & Cross-Axis Grid Alignment, Custom Dialog Prompts, and Multi-View Route Transitions (`Navigator` management).
